# 飞书多维表格配置指南

## 创建多维表格

1. 打开飞书，进入「多维表格」
2. 新建一个表格，命名为「会议待办跟踪」
3. 按下方字段结构创建列

## 字段结构

| 字段名 | 类型 | 说明 |
|--------|------|------|
| ID | 文本 | 待办编号，格式 M{场次}-{序号}，如 M1-01 |
| 待办事项 | 文本 | 待办标题 |
| 负责人 | 单选 | 添加团队成员作为选项 |
| 截止日期 | 日期 | 截止时间 |
| 完成标准 | 文本 | 产出物 + 验收人 |
| 验收人 | 文本 | 谁来确认完成 |
| 协同方 | 文本 | 需要配合的人 |
| 来源会议 | 文本 | 会议名称和日期 |
| 状态 | 单选 | 待确认 / 进行中 / 已完成 / 已取消 |

## 获取配置信息

创建好表格后，从 URL 中获取两个值：

```
https://xxx.feishu.cn/base/ABCdef123?table=tblXYZ456
                           ^^^^^^^^       ^^^^^^^^^
                           base_token     table_id
```

把这两个值填入 `config.yaml`：

```yaml
feishu:
  bitable:
    base_token: "ABCdef123"
    table_id: "tblXYZ456"
```

## 权限设置

确保你的飞书应用有以下权限：
- `bitable:app` — 多维表格读写
- `docs:doc` — 文档创建
- `im:message` — 发送消息（如果用飞书 IM 通知）

在飞书开放平台 → 你的应用 → 权限管理中开启。
